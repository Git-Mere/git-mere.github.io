Long ChatGPT conversations make it tedious to scroll back to earlier questions, causing users to lose context and waste time. This Chrome extension automatically places color-coded markers beside the scrollbar at each question's position, with hover previews and click-to-navigate support across ChatGPT, Gemini, Claude, and Grok. A favorites system lets users right-click any marker to pin important questions, and the popup panel provides a full list view for direct navigation. Published on the Chrome Web Store, it removes scroll friction from long AI chat sessions without altering the host page's interface.
